# Super Resolution in OpenCV


## Instructions

### Python

To run the code in Python, please use `super_res.py`

### C++

To run the code in C++, please follow the steps given below:

```
mkdir build
cd build
cmake ..
cmake --build . --config Release
cd ..
./build/SuperResolution
```
